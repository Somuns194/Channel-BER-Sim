# Channel-BER-Sim
Lab 1 MATLAB Simulation: BER vs SNR for Multiple Modulation under AWGN & Rayleigh Channel

## Project Introduction
This is the first lab assignment of my Wireless Mobile Communications course. I wrote a complete Monte Carlo simulation code to calculate bit error rate of five mainstream modulation schemes, including QPSK, 8PSK, 16PSK, 16QAM and 32QAM.

Two common wireless channel models are built separately for comparison: ideal AWGN channel and Rayleigh flat multipath fading channel with ZF equalization. I drew simulated BER curves and compared them with theoretical values calculated by MATLAB built-in functions, so as to analyze the anti-noise and anti-fading ability of different modulations.

Four reference papers are attached in the repo, which supply theoretical support for BER formula deduction and ZF equalizer performance analysis. The complete lab report records all mathematical derivation, simulation steps, curve comparison and actual debugging problems I met during coding.

## Experiment Content
1. Build two independent, reusable simulation modules for AWGN and Rayleigh flat fading channel.
2. Complete Gray mapping and hard demodulation logic for QPSK, 8PSK, 16PSK, 16QAM and 32QAM.
3. Traverse the whole SNR range, accumulate error bits through Monte Carlo circulation, and draw BER-SNR simulation curves.
4. Calculate theoretical BER values by official communication functions, overlay simulated curves and theoretical curves to do visual contrast analysis.
5. Add ZF equalization process for Rayleigh fading channel, and analyze the noise amplification defect brought by equalization.

## Debug Notes & Optimization
1. At the beginning, simulated BER curves deviated a lot from theoretical results. After checking codes, I fixed the power normalization factor of Rayleigh fading channel, and the curves matched theory well.
2. The program ran very slowly at low SNR because of too many transmitted symbols. I optimized the simulation efficiency by reducing total symbols in low SNR intervals.
3. BER data fluctuated sharply at high SNR with few error bits counted. I raised the minimum statistical error bit threshold to make curves smooth and reliable.
4. Unit conversion between Es/N0 and Eb/N0 was missing at first, which caused wrong theoretical curves. I added corresponding conversion formulas for different modulation orders.
5. Added `.gitignore` file to avoid uploading large temporary cache files generated by MATLAB.

## Core Experiment Conclusions
1. Under the same AWGN SNR condition, QAM modulation has better error performance than PSK with the same spectrum efficiency. Higher modulation order will lead to higher bit error rate, as the distance between constellation points becomes shorter.
2. Rayleigh multipath fading causes obvious BER deterioration compared with ideal AWGN channel. Even after ZF equalization, noise amplification will form error floor at high SNR, which cannot be eliminated by simply increasing transmitting power.
3. The performance order of the five modulations stays consistent in both channel environments: QPSK > 16QAM > 8PSK > 32QAM > 16PSK.
4. Take 16QAM as an example: to reach BER=1e-3 under Rayleigh fading, extra 12dB signal power is needed compared with AWGN channel, which reflects the huge performance loss caused by wireless multipath fading.

## How to Run
1. Open the corresponding `.m` script in MATLAB R2023b or above version.
2. Adjust SNR range and total simulation symbols at the top of the code according to your demand.
3. Click run to start Monte Carlo simulation; real-time BER data will be printed in command window.
4. After simulation finishes, semi-log BER comparison graph will pop up automatically.
